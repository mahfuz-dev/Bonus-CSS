CSS Position - 'Static' , 'Relative' , 'Absolute' , 'Fixed'

position-relative

By default each of the div's position is 'Static'

If we fix the position of a div in 'Relative' then it's changable from it's main position.By using 'left','right','top','bottom' on the element position can be changed

position- absolute

By Absolute the position will be changed compares to the Parent tag/class.Here the parent is 'container'.But here the initial position of the element will be vanished permanently after changing the position with 'absolute'

position- fixed

The 'position:fixed;' will make an element fixed to it's position through the whole page.

CSS specificity style order style priority

CSS styling has priority order we know how it works.

Inline CSS is the most powerful over everything.But we can beat that in external CSS by adding '!important' beside the style action even there is other style in the inline css of that element

Example:h1{ color:red !important; }

If there are multiple '!important'(CSS specificity) in css on the same tag's styling.Then the style will be applied according to the priority order. But CSS specificity is not recommended to use
